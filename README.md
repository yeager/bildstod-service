# Bildstöd — Swedish Pictogram Library

Free Swedish pictograms for NPF/autism, school, and Swedish culture. ARASAAC-style, CC BY-SA 4.0.

## Features

- 16 pictograms (and growing) in SVG + PNG format
- Focus on Swedish/Nordic concepts missing from ARASAAC
- NPF/autism-specific pictograms (visual support, calm corner, sensory break, etc.)
- JSON API for integration
- Search and category filtering
- Hosted at [autismappar.se/bildstod](https://autismappar.se/bildstod)

## Categories

- **Svensk kultur**: fika, kanelbulle, semla, dalahäst, lucia, midsommar, kräftskiva
- **NPF / Autism**: bildstöd, lugna hörnan, hörselkåpor, tyngdtäcke, sensorisk paus, social berättelse, schema
- **Skola**: förskola, fritids

## API

```
GET /bildstod/api/pictograms.json
```

Returns all pictograms with keywords, categories, descriptions and file paths.
